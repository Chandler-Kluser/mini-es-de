# Griddlemethis-DE for EmulationStation Desktop Edition

Griddlemethis-DE is a theme set for ES-DE that supports all systems as well as the latest application features.

![systems](sys.png)
![games](game.png)

The following options are included:

2 variants:

- Grid with videos
- Grid without videos

7 color schemes:

- Chilled
- Gold
- Greyscale
- Retro
- Purple Rain
- Primary
- Stay Classy


4 aspect ratios:

- 16:9
- 4:3
- 16:10
- 21:9
